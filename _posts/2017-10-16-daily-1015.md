阿里云免费签发的CA证书，然后下载到服务器，配置nginx

```
  ssl_certificate   cert/213970811020013.pem;
  ssl_certificate_key  cert/213970811020013.key;
```

对80端口的域名访问做好跳转
```
server {
	listen 80;
    server_name server.com www.server.com;
    return 301 https://www.server.com$request_uri;
}

```
